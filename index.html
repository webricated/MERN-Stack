Web Development Syllabus

Web Development Fundamentals

  • Understand how websites work and how HTML, CSS and JavaScript contribute.
  • Understand how the internet works and how websites are served.
  • Learn to use git, GitHub and version control.
  • Learn key troubleshooting and debugging skills to apply to your projects.

HTML 5

  • Learn the anatomy of HTML syntax to structure your websites.
  • Understand the HTML boilerplate and HTML doctypes.
  • Understand indentation and nesting in HTML code.
  • Use HTML tags for headings and paragraphs.
  • Structure unordered and ordered lists.
  • Insert images and create hyperlinks.
  • Create multi-page websites.
  • Learn HTML best practices.

CSS 3

  • Understand cascading style sheets (CSS) and how to style your site.
  • Use selectors, properties, and understand inline, internal, and external CSS.
  • Understand specificity, inheritance, and the CSS Box Model.
  • Use developer tools for CSS debugging.
  • Apply positioning, display, font styling, and media queries.

Flexbox

  • Learn why Flexbox was introduced and how it works.
  • Use flex direction, align, justify, and flex shorthand properties.

Grid

  • Learn how CSS Grid differs from Flexbox.
  • Create and arrange grids and combine Grid with Flexbox.

Web Design

  • Learn web design principles, UI/UX basics, color theory, and typography.
  • Apply attention management principles for better UX.

JavaScript ES6

  • Understand syntax, data types, operators, control structures, functions, arrays, and objects.
  • Work with higher-order functions, array/object methods, OOP concepts.
  • Use modern ES6 features like arrow functions, spread/rest operators, and destructuring.

DOM

  • Learn HTML tree structure, DOM traversal, and manipulation using JavaScript.

Bootstrap

  • Understand native CSS vs. external frameworks.
  • Use Bootstrap grid, components, icons, and templates.

Version Control

  • Use Git and GitHub for version control, branching, rolling back changes.

Unix Command Line

  • Use basic bash commands to manage files and folders.

Node.js

  • Learn back-end JavaScript development, Node basics, NPM, events, and file system.

Express.js

  • Install and use Express, RESTful routing, middleware.

EJS

  • Use EJS templating with Node/Express, pass data, and structure layouts.

APIs

  • Understand APIs, HTTP, authentication, and JSON data transmission.

Databases

  • Understand SQL, CRUD operations, Joins, and schema design.
  • Work with PostgreSQL and MongoDB, ER models and ORM concepts.

Authentication & Security

  • Understand encryption, hashing, sessions, and OAuth 2.0 for secure user auth.

React.js

  • Learn React components, JSX, props, state, hooks, and conditional rendering.

Web3 & Blockchain

  • Understand tokens, NFTs, wallets, and deploy projects on the Internet Computer using blockchain.
